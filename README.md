# lifecountdown

Life Countdown は、ブラウザの新しいタブを「人生の残り時間」を可視化するダッシュボードに変える Chrome 拡張です。
生年月日と寿命の想定年数から「人生の残り時間」を計算し、円グラフとカウントダウンで表示します。また、毎日 8 時間の睡眠時間を自動的に差し引き、「実際に活動できる残り時間（アクティブな時間）」だけを年・日・時刻、さらには「xxxxh yyyy sec」という形で確認できます。
残り人生の割合を円グラフで可視化：経過した時間と残り時間を一目で把握
詳細なカウントダウン表示：残り「年・日・時・分・秒」をリアルタイムで更新
活動時間ベースの残り時間：睡眠 8 時間/日を差し引いた「使える時間」だけを計算
新しいタブをライフダッシュボード化：日々ブラウザを開くたびに、時間の貴重さをリマインド
「なんとなく過ぎていく毎日」を、「限られた活動時間を意識して生きる」感覚に切り替えるための、シンプルなライフログ系ツールです。
